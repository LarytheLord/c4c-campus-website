## Key Concepts

### Two Types of Discussions

**Lesson Discussions:**
- Specific to one lesson
- Threaded (nested replies)
- For questions about lesson content

**Course Forums:**
- Course level (not lesson-specific)
- Can be locked
- For general Q&A, projects, announcements

### Teacher Responses

When a teacher replies, they get a special ğŸ‘¨â€ğŸ« badge showing it's authoritative.
