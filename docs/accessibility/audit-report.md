## Common Issues & Fixes

### Color Contrast
- Ensure text has at least 4.5:1 contrast ratio
- Large text (18pt+/14pt+ bold) needs at least 3:1
- Use tools like WebAIM Contrast Checker

### Keyboard Navigation
- All interactive elements must be keyboard accessible
- Provide visible focus indicators
- Ensure logical tab order

### ARIA Labels
- All form inputs need associated labels
- Buttons need descriptive accessible names
- Use ARIA landmarks for page structure

### Images
- Provide alt text for content images
- Use empty alt="" for decorative images
- Describe the function for linked images
