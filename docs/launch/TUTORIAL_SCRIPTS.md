## Video Recording Guidelines

If recording these tutorials:

### Equipment Needed
- Screen recording software (OBS, ScreenFlow, Camtasia)
- Microphone (even built-in is fine)
- Optional: webcam for intro/outro

### Recording Tips

1. **Audio**
   - Speak clearly and at a natural pace
   - Leave pauses between sections
   - Use a quiet room with minimal background noise
   - Test audio levels before recording

2. **Screen**
   - Use 1920x1080 resolution minimum
   - Zoom browser to 125% for readability
   - Highlight UI elements as mentioned
   - Move cursor slowly and deliberately

3. **Pacing**
   - Allow 2-3 seconds after each action
   - Pause before important information
   - Don't rush - viewers can't skip ahead
   - Add visual transitions between sections

4. **Editing**
   - Cut out dead air and mistakes
   - Add title cards for each section
   - Use background music (royalty-free, low volume)
   - Add captions for accessibility
   - Export at 1080p minimum

### Publishing

- **Platform**: YouTube (recommended), Vimeo, or in-platform videos
- **Captions**: Required for accessibility
- **Descriptions**: Include timestamps and links
- **Thumbnails**: Consistent branding with C4C logo
- **Playlists**: Group by role (Student, Teacher, Admin)
